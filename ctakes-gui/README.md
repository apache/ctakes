# cTAKES GUI

Contains code used for cTAKES GUI tools.

The primary GUI tools are:
* [Piper File Submitter](https://github.com/apache/ctakes/wiki/Piper-File-Submitter)
* [Dictionary Creator](https://github.com/apache/ctakes/wiki/Dictionary-Creator)
* [UMLS Package Fetcher](https://github.com/apache/ctakes/wiki/cTAKES-UMLS-Package-Fetcher)
* [Simple Pipeline Fabricator](https://github.com/apache/ctakes/wiki/Simple-Pipeline-Fabricator)
